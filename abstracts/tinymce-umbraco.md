# Tiny but mighty: TinyMCE in Umbraco

While TinyMCE is no longer Umbraco’s default rich text editor, you can still easily use it to satisfy all your Umbraco text editing needs. In this talk, Fred will show you how to get the ball rolling with the TinyMCE Package to keep using the WYSIWYG you love without a fuss, and more importantly, without losing data in the process.
