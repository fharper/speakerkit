# Shipping happiness with Umbraco packages

Umbraco’s ecosystem thrives on packages, but poorly built and maintained ones create frustration for both developers and editors. In this talk, we’ll explore how to build open source Umbraco packages from scratch, while being mindful of everything else needed to ensure the best experience for everyone. From best practices, user documentation and versions compatibility, to developer support, external contributors’ management and helpful automations, you will learn how to extend the friendly CMS functionalities while having fun, and giving back to the community.
